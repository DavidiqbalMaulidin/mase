🚀 MASE QUICK START — 5 Minutes Setup
====================================

# ⚡ FASTEST WAY TO GET STARTED

## Step 1: Install Dependencies (30 seconds)
```bash
npm install
```

## Step 2: Verify Environment Variables (10 seconds)
Check `.env.local` has these variables:
- NEXT_PUBLIC_SUPABASE_URL ✓
- NEXT_PUBLIC_SUPABASE_ANON_KEY ✓

They are already configured for you!

## Step 3: Setup Database (1 minute)
```bash
npm run db:setup
```
This creates tables in Supabase automatically.

## Step 4: Start Development Server (30 seconds)
```bash
npm run dev
```
Visit: http://localhost:3000

## Step 5: Test the App (2 minutes)
1. Go to http://localhost:3000 (see beautiful landing page)
2. Click "Get Started" button
3. Register a new account
4. Add some transactions in dashboard
5. Check analytics charts

# 🆘 TROUBLESHOOTING

### "npm: command not found"
→ Install Node.js from https://nodejs.org

### "NEXT_PUBLIC_SUPABASE_URL not found"
→ Check .env.local file exists with variables

### "Database tables not created"
→ Run: npm run db:setup

### "Port 3000 already in use"
→ Run: npm run dev -- -p 3001

### "Localhost not loading"
→ Wait 30 seconds for build
→ Check console for errors
